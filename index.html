Palindrome Partitioning (C)

This program takes an input string and determines the minimum number of palindromes needed to create the string. For example, "racecar" is a palindrome, so the program will return 1. "abcde" consists of 5 unique characters, and a character itself is a palindrome, so 5 will be returned.

The file contains 2 different implementations of the program. The first is a recursive solution, and the second is a dynamic programming solution. When comparing the running times, it becomes clear that the dynammic programming solution is vastly more efficient when the input string becomes sufficiently long.
